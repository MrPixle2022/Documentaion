# OS Module

---

### to use it:

```javascript
import os from 'os';
```

---

#### platform:

```javascript
console.log(os.platform());
//win 23
```

the [**platform**] method returns the name of the os platform

---

#### arch:


```javascript
console.log(arch());
//x64
```

the [**arch**] method returns the architecture of the os

---

#### cpus:

```javascript
console.log(is.cpus());
/*

```

the [**cpus**] method returns data about the cpu like model, etc
